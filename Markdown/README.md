Markdown语法整理
===
一、区块元素
---
### 1、标题

* 使用 ` = ` (最高标题)和 ` - ` (第二阶标题)
    
        a
        =
        b
        -
    >其中 ` = `  ` - ` 至少出现一次  
    >标题下会自动添加分割线
* 使用 ` = `

        # 1
        ## 2
        ### 3
        #### 4
        ##### 5
        ###### 6
    >` # ` 的数量可以为1~6个，` # ` 后面要有空格  
    >一个 ` # ` 相当于 ` = ` ，两个 ` # ` 相当于 ` - `  
    >可以选择闭合，在行尾添加 ` # ` ，数量可以与前面的不一致

### 2、段落和换行
* 空行被视为换行
* 行尾添加两个以上（包括两个）来换行
* 普通段落不该用空格或制表符来缩进
